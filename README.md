SGS — National Sports Management System

A modern frontend prototype for a National Sports Management System, designed to manage sports infrastructure, events, documents, and workflows with integrated analytics and AI features.

Overview

SGS provides a centralized platform for managing national sports operations, including scheduling, document automation, infrastructure monitoring, and decision-making tools powered by data and AI.

Tech Stack
HTML5
Tailwind CSS (CDN)
Vanilla JavaScript (Modular)
Chart.js — analytics & dashboards
Leaflet.js — GIS & map visualization
▶ Running the Prototype
Option 1 — Direct

Open index.html in a modern browser

Option 2 — Local server (recommended)
npx serve .
# or
python -m http.server 8000
